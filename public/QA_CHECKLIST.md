# MiniMarket QA Checklist (v5)

## 1) 탐색/검색
- [ ] Ctrl+K로 검색 input 포커스 이동
- [ ] 자동완성: 최근검색어/추천검색어 노출 조건
- [ ] 자동완성 키보드: ↑/↓ 이동, Enter 선택, Esc 닫기
- [ ] 최근검색어 전체삭제 동작
- [ ] 검색어 입력 시 URL에 q 반영, 새로고침해도 유지

## 2) 카테고리/메가메뉴
- [ ] 상단 '전체 카테고리' 클릭 → 오버레이 열림
- [ ] 마우스 hover로 오른쪽 추천 영역 변경
- [ ] Enter 키로 카테고리 선택 가능
- [ ] Esc/바깥 클릭으로 닫기
- [ ] 포커스 트랩(탭 이동이 dialog 내부에서만 순환)

## 3) 필터/정렬/페이지/뷰(상태 유지)
- [ ] 가격 최소/최대 입력 경계: 0, 음수, 문자 입력(브라우저마다 다름)
- [ ] 평점 필터: 4.5/4.0/3.5 기준 포함 여부
- [ ] 배송 타입 필터: rocket/dawn/normal
- [ ] 무료배송 필터: 49,999 vs 50,000 경계
- [ ] 재고있는 상품만 체크 시 품절 제외
- [ ] 정렬(가격/평점/리뷰) 동작 및 URL 반영
- [ ] 페이지네이션 이동 시 URL page 유지
- [ ] 그리드/리스트 전환 시 URL view 유지

## 4) 상품 카드/쿠폰/할인
- [ ] 할인 표시: 원가 취소선 + 할인 적용가 노출
- [ ] 쿠폰 버튼 클릭 → 토스트(목업) 노출
- [ ] 품절 상품 담기 버튼 disabled
- [ ] 재고 1 상품 뱃지(재고1) 노출

## 5) 찜(위시리스트)
- [ ] 카드 하트 토글, 토스트 메시지, 상태 유지(localStorage)
- [ ] 찜 페이지(wishlist.html)에서 목록/empty 상태
- [ ] 찜 삭제 후 목록 즉시 반영

## 6) 최근 본 상품
- [ ] 상품 상세 진입 시 최근본에 추가(중복 제거, 최대 12개)
- [ ] 홈에 최근본 스트립 노출/empty 상태
- [ ] 최근본 지우기 동작

## 7) 옵션 모달(의류/신발)
- [ ] 의류/신발: 옵션(사이즈/색상) 선택 UI 노출
- [ ] 옵션 없는 상품: 안내 문구 노출
- [ ] 수량 입력: 음수/0/재고 초과 처리
- [ ] Esc/바깥 클릭으로 닫기, 포커스 트랩 동작
- [ ] 같은 상품 다른 옵션을 각각 담을 수 있는지(라인 분리)

## 8) 장바구니/결제
- [ ] 장바구니: 옵션 표시, 할인 단가/합계 반영
- [ ] 무료배송 경계에서 배송비 계산
- [ ] 주문하기: 로그인 유도(미로그인)
- [ ] 결제 Mock 성공/실패 흐름
- [ ] 주문내역 저장/표시

## 9) 비정상/복원
- [ ] 새로고침 시 URL 상태 + localStorage 상태 동기화
- [ ] localStorage 초기화 후 동작(README 참고)
